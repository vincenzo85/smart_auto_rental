## GitHub Pages

La pagina pronta per pubblicazione e `docs/index.html`.

Per usarla su GitHub Pages:

1. Push del branch remoto con i file aggiornati.
2. In GitHub: `Settings > Pages`.
3. In `Build and deployment`, seleziona `Deploy from a branch`.
4. Scegli branch `main` (o il branch pubblicato) e cartella `/docs`.
5. Salva e attendi la pubblicazione dell'URL Pages.
